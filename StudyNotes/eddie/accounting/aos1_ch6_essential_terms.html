🧮 Key Investment Formulas ▼

Rate of Return

Net return / Cost x 100

Tells you what percentage return you earned on your investment.

Invested $5,000, net return $400
= 400 / 5,000 x 100 = 8% p.a.

Simple Interest

P x R x T / 100

P = Principal, R = Rate (%), T = Time (years). Interest calculated on original amount only.

$10,000 at 5% for 3 years
= 10,000 x 5 x 3 / 100 = $1,500

Compound Interest

P x (1 + R/100)^T - P

Interest calculated on the growing balance — you earn interest on your interest.

$10,000 at 5% for 3 years
= 10,000 x (1.05)^3 - 10,000 = $1,576.25

Net Return

Gross return - Costs

The actual profit from an investment after deducting all associated costs and fees.

Dividend $600, brokerage fee $50
Net return = $600 - $50 = $550

💡 Always use NET return (after costs) when calculating rate of return in VCE — a very common exam mistake.
